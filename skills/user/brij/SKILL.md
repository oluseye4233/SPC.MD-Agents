## Primary Workflow

BRIJ executes across four integration layers.

### Layer 1 — Merchant Onboarding & Catalog Integration
*Single integration setup, product feed normalization, platform certification*

**Merchant Setup Protocol:**
- API credential provisioning for Google UCP, Microsoft Copilot, OpenAI ChatGPT
- Product catalog ingestion: normalize from merchant format to universal BRIJ schema
- Category mapping: merchant taxonomy → Google Product Categories / Microsoft Catalog / OpenAI product schema
- Pricing rules: base price, discount tiers, promotional pricing with platform-specific overrides
- Inventory integration: real-time stock levels, fulfillment location mapping, lead time data
- Media assets: product images, videos, 3D models — optimized per platform requirements

**BRIJ Universal Product Schema:**
```yaml
BRIJ_PRODUCT:
  product_id: [merchant_sku]
  title: [normalized, keyword-optimized]
  description: [platform-adaptive text]
  price: {base, sale, currency}
  inventory: {quantity, location, fulfillment_time}
  media: {primary_image, gallery, video}
  categories: {merchant, google, microsoft, openai}
  attributes: [platform-specific metadata]
```

### Layer 2 — Platform Protocol Adapters
*Google UCP, Microsoft Copilot, OpenAI ChatGPT platform-specific compliance*

**Google Universal Commerce Protocol (UCP) Adapter:**
- Product feed: Google Merchant Center format, Shopping Actions compliance
- Checkout integration: Google Pay API, Google Checkout flow
- Conversational triggers: Google Assistant commerce intents
- Analytics: Google Analytics 4 commerce events

**Microsoft Copilot Checkout Adapter:**
- Copilot Commerce API integration
- Microsoft Pay checkout flow
- Bing product listing compliance
- Teams commerce channel configuration

**OpenAI ChatGPT Instant Checkout Adapter:**
- ChatGPT plugin/connector configuration
- Instant Checkout flow implementation
- Conversational product discovery schema
- OpenAI payment processing integration

### Layer 3 — Intelligent Routing & Optimization
*Platform performance routing, price discovery, payment optimization*

**Automatic Platform Routing:**
- User platform detection: which AI platform is the consumer using?
- Performance routing: route to best-performing platform based on historical conversion data
- Price discovery: identify best available pricing across all platforms for consumer
- Payment method optimization: match consumer preferred payment to platform capabilities

**Cross-Platform Consumer Profile:**
- Unified purchase history: aggregate orders across all three platforms
- Preference learning: product categories, price sensitivity, preferred payment method
- Cross-platform loyalty: rewards valid across Google, Copilot, and ChatGPT transactions

### Layer 4 — Merchant Dashboard & Analytics
*Unified reporting, performance optimization, inventory management*

**Unified Dashboard:**
- GMV breakdown by platform (Google/Copilot/ChatGPT)
- Conversion rates by platform and product category
- Payment method distribution and success rates
- Inventory status across all platforms in real time

**Performance Analytics:**
- Platform comparative analysis: which AI platform drives highest conversion for this merchant/product?
- Routing effectiveness: was automatic routing optimal?
- Consumer behavior: journey from discovery to purchase across platforms
- Fraud detection: cross-platform anomaly detection, dispute management
