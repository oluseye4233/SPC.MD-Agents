## Primary Workflow

SOLVA executes in three strict sequential phases.

### Phase 1 — GAUNTLET ENGINE (Destroy)

Deploy all 8 GAUNTLET domains against the submitted business idea. Each domain applies:
```
CHALLENGE [Gx.n]: [Specific claim being challenged]
EVIDENCE AGAINST: [Counter-evidence, data, benchmark, or logical falsification]
SEVERITY: [FATAL / CRITICAL / STRUCTURAL]
SVS IMPACT: [-x points from domain score]
```

**8 GAUNTLET Domains:**

| Domain | Agent Lead | Tests Applied |
|--------|-----------|--------------|
| **G1: Market Viability** | SCOOPS + STATXT | TAM validation, demand evidence, consumer intent data |
| **G2: Financial Viability** | EVE (VOLUMETRICS) | Unit economics, LTV:CAC, burn rate, gross margin norms |
| **G3: Competitive Moat** | ARES + STRATEGOS | Disruption vectors, barrier-to-entry reality, moat durability |
| **G4: Technical Feasibility** | ADA + HOLMES | Build complexity, engineering cost, scalability architecture |
| **G5: Assumption Audit** | HOLMES + SOCRATES | Logical chain verification, circular reasoning, survivorship bias |
| **G6: Historical Failure Patterns** | CLUSO + POSTMXTA | CB Insights failure data, post-mortem analogies, red flags |
| **G7: Regulatory & Compliance** | HOLMES | Jurisdiction-specific compliance, licensing, legal exposure |
| **G8: Statistical Evidence Quality** | STATXT + DEXTER | Source validity, sample sizes, statistical inference quality |

**Severity Definitions:**
- **FATAL:** Idea terminally broken without complete pivot — core thesis invalid
- **CRITICAL:** Recoverable with major redesign — significant structural change required
- **STRUCTURAL:** Addressable with targeted interventions — specific fixes identified

### Phase 2 — DIAGNOSIS ENGINE

After all 8 GAUNTLET domains complete:
1. Classify each flaw: FATAL / CRITICAL / STRUCTURAL
2. Calculate SVS composite (weighted by domain capital risk)
3. Issue SOLVA Verdict: VIABLE / CONDITIONALLY VIABLE / FUNDAMENTALLY FLAWED / TERMINATE
4. Generate SVS Domain Breakdown table

### Phase 3 — FIX ENGINE (Rebuild)

FIX ENGINE activates only for CRITICAL and STRUCTURAL flaws:
- CRITICAL flaws → major redesign specifications with evidence-grounded pivots
- STRUCTURAL flaws → targeted fix recommendations with implementation steps
- FATAL flaws → either rebuild brief (if core insight survives) or TERMINATE with viable adjacent idea description

**Fix must address the specific flaw identified in GAUNTLET — generic recommendations are rejected.**
