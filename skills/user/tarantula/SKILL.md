## Primary Workflow

TARANTULA executes the **FORGE 7-Phase Pipeline** for production website delivery.

### Phase 1 — Discovery (Requirements Analysis)

**Objective:** Comprehensive requirements gathering and validation

- Parse input requirements specification (business, technical, content, marketing objectives)
- Identify gaps/ambiguities → generate prioritized clarifying questions
- Validate technical feasibility against 4J.BONSAI capabilities
- Map content needs to SHAY content generation patterns (page count, word targets, tone spec)
- Align marketing objectives with SCOOPS pull strategy hooks
- **Deliverable:** Requirements Validation Report (RVR)
- **Success Criteria:** 100% fields validated, zero critical ambiguities, feasibility ≥90%

### Phase 2 — Design (Architecture & Wireframing)

**Objective:** Comprehensive design blueprint

4J.BONSAI Integration:
- Information architecture mapping (sitemap, navigation hierarchy, page relationships)
- Component library selection (shadcn/ui patterns, custom components needed)
- Responsive breakpoint strategy (mobile-first: 375px, 768px, 1024px, 1280px, 1536px)
- Performance optimization planning (Core Web Vitals targets: LCP <2.5s, CLS <0.1, FID <100ms)
- **Deliverable:** Design System Document (DSD) with annotated wireframes

### Phase 3 — Development (Code Generation)

**Objective:** Build production-ready website code

4J.BONSAI Technical Stack implementation:
- Next.js 14+ App Router with React Server Components for optimal performance
- Tailwind CSS utility-first styling with custom design tokens
- Framer Motion for micro-interactions and page transitions
- React Hook Form + Zod for type-safe form validation
- Image optimization via Next.js Image component with WebP/AVIF formats
- **Quality Gates:** Lighthouse ≥90, mobile verified, cross-browser confirmed, zero critical CVEs

### Phase 4 — Content Population (SHAY Integration)

**Objective:** Generate and deploy compelling, SEO-optimized content

SHAY Content Engine:
- Brand voice extraction and consistency rules established
- Homepage copy: hero, value proposition, social proof, CTA hierarchy
- Product/service pages: feature explanations, benefit statements, FAQ sections
- Blog/resource content: SEO-targeted articles with keyword clusters
- Metadata: title tags (50-60 chars), meta descriptions (150-160 chars), OG tags
- Structured data: JSON-LD schema for organization, product, article, FAQ
- Internal linking: topic cluster architecture with pillar/spoke structure

### Phase 5 — Marketing Integration (SCOOPS Activation)

**Objective:** Embed 5 pull marketing hooks for organic traffic generation

| Hook | System | Implementation |
|------|--------|---------------|
| Hook 1: SEO Foundation | Technical SEO | XML sitemap, robots.txt, canonical URLs, structured data, Open Graph |
| Hook 2: Lead Capture | Multi-stage funnel | Exit-intent popups, newsletter signups, content upgrades, consultation booking |
| Hook 3: Social Proof Engine | Trust architecture | Testimonials, case studies, trust badges, review aggregation, client logos |
| Hook 4: Content Marketing Hub | Resource center | Blog, downloadable guides, video library, webinar registration, interactive tools |
| Hook 5: Conversion Optimization | Psychological triggers | Scarcity, urgency, social proof, A/B framework, heatmap tracking, form optimization |

### Phase 6 — Testing & Optimization

**Objective:** Ensure flawless cross-platform performance

- Lighthouse audit across all 4 dimensions (Performance, Accessibility, Best Practices, SEO)
- Cross-device testing: mobile, tablet, laptop, desktop (5 breakpoints)
- Cross-browser testing: Chrome, Firefox, Safari, Edge
- Form validation testing (happy path + error states)
- Link validation (zero broken links)
- Content review: grammar, brand voice consistency, CTA clarity
- Security scan: Content Security Policy, XSS protection, HTTPS enforcement

### Phase 7 — Deployment & Monitoring

**Objective:** Launch with continuous improvement systems

- **Deployment:** Vercel (default) with CI/CD pipeline, preview deployments for PRs
- **Uptime Monitoring:** UptimeRobot/Pingdom (99.9% SLA target)
- **Error Tracking:** Sentry + LogRocket (session replay)
- **Performance:** Vercel Analytics + Google Analytics 4 (Core Web Vitals monitoring)
- **SEO:** Google Search Console + Ahrefs rank tracking
- **Conversion:** Goal completion, heatmaps, session recordings
- **Deliverable:** Production-ready deployment + monitoring dashboard
