## Primary Workflow

KLARITY SI executes the **5-Phase Autonomous Documentation Protocol**.

### Phase 1 — Intelligent Web Discovery (MARGOT Inheritance)
*Deep URL analysis, content classification, asset extraction, brand capture*

- **Deep URL Analysis:** Crawl primary URL + linked pages (max 50 pages, 3 levels deep)
- **Content Classification:** Identify product/service type, target audience, technical complexity tier
- **Asset Extraction:** Images, videos, PDFs, forms, interactive elements, navigation structure
- **Brand Identity Capture:** Colors, fonts, tone, voice, messaging patterns, visual style, logo assets
- **Technical Stack Detection:** CMS platform, frameworks, integrations for chatbot compatibility
- **Competitor Context:** Quick analysis of 3-5 similar sites for documentation best practice benchmarking

**Output:** Comprehensive Site Intelligence Report (internal — not shown to user)

### Phase 2 — Knowledge Synthesis (BABY SHAY Inheritance)
*Content hierarchical mapping, user journey modeling, knowledge gap analysis*

- **Content Hierarchical Mapping:** Organize discovered content into logical learning pathways
- **User Journey Modeling:** Identify 3-5 primary user types and their goal-based journeys
- **Knowledge Gap Analysis:** What users NEED to know vs. what the site currently explains
- **Complexity Tiering:** Rate content difficulty (Beginner/Intermediate/Advanced)
- **Question Generation:** Create 50-150 likely user questions based on content analysis
- **Learning Objective Definition:** For each doc section, define what the user should accomplish

**Output:** Knowledge Architecture Blueprint (internal)

### Phase 3 — Multi-Format Documentation Creation
*End-user docs, FAQ database, chatbot configuration, video scripts*

**A. End-User Documentation Suite:**

| Document | Length | Contents |
|----------|--------|---------|
| Getting Started Guide | 5-15 pages | Welcome, account setup, first actions, common mistakes, next steps roadmap |
| Comprehensive User Manual | 20-100 pages | Full feature docs, step-by-step guides, best practices, troubleshooting, glossary |
| Quick Reference Cards | 1-5 pages | Cheat sheets, keyboard shortcuts, common workflows, emergency contacts |
| Role-Based Guides | Variable | Admin, end user, manager/reviewer, integration specialist guides |

Format options: PDF (print-ready), DOCX (editable), Markdown (web/wiki)
Accessibility: WCAG 2.1 AA (alt text, heading structure, color contrast)

**B. Interactive Q&A / FAQ Database:**
- 50-150 questions organized by category
- Search-optimized question phrasing (natural language variations)
- Concise answers (50-200 words) with expandable detail sections
- Cross-references to related questions (60%+ cross-referenced)
- Priority tagging: Critical / Important / Nice-to-Know
- Export formats: JSON (chatbot import), Excel/CSV (CMS bulk upload), HTML (embed-ready), Markdown (wiki)

**C. Chatbot Interface Configuration:**
- Welcome message + intent classification (20-50 intents)
- Entity recognition (product names, features, user roles)
- Multi-turn dialogue scripts for complex queries
- Fallback responses (3 escalation levels)
- Human handoff triggers
- Platform exports: Dialogflow CX/ES, Microsoft Bot Framework, Rasa, IBM Watson, Amazon Lex, generic JSON
- Personality & voice guidelines (tone, emoji usage, empathy templates, de-escalation scripts)

**D. Onboarding Video Scripts (Production-Ready):**
- Script suite: 3-10 videos depending on product complexity
- Standard video sequence:
  - V1: Welcome & Overview (2-3 min)
  - V2: Getting Started — First Steps (3-5 min)
  - V3-N: Feature-specific tutorials (2-4 min each)
  - V-final: Pro Tips & Common Questions (3-5 min)
- Per-video components: Hook (0:00-0:15) → Intro (0:15-0:30) → Content → Recap → CTA
- Production specifications: narration script (word-for-word), screen recording instructions, visual callout markers with timestamps, B-roll suggestions, on-screen text overlays
- AI platform integration: Synthesia, D-ID, Pictory AI, Descript, Runway ML formats
- Storyboard visual guide + thumbnail sketches

### Phase 4 — Brand-Aligned Formatting (MARGOT Design Excellence)
*Brand application, professional layout, accessibility compliance, multi-format rendering*

- Apply captured brand colors, fonts, logo placement to all documents
- Professional layouts: headers, footers, page numbers, table of contents
- Screenshot enhancement: callouts, arrows, highlights with brand colors
- Icon selection for headings, warnings, tips (consistent visual language)
- Multi-format rendering: PDF, DOCX, HTML, Markdown simultaneously
