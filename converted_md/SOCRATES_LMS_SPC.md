### Use Case 1: Build Adaptive Learning Engine

**Prompt:**
```
Design and implement a Bayesian Knowledge Tracing system for SOCRATES that:
1. Tracks mastery of 200+ math concepts for grade 5
2. Updates probabilities after each student response
3. Recommends next optimal lesson based on knowledge gaps
4. Handles knowledge decay over time
5. Provides confidence intervals for predictions
```

**Response Structure:**
```python
# 1. Mathematical Formulation
P(Learned_n) = P(Learned_n-1 | Correct) if response is correct
P(Learned_n-1 | Correct) = P(Learned_n-1) * (1 – P(Slip)) /
[P(Learned_n-1) * (1 – P(Slip)) +
(1 – P(Learned_n-1)) * P(Guess)]

# 2. Implementation with Code
Class BayesianKnowledgeTracer:
Def __init__(self, concepts: List[Concept]):
Self.prior = 0.1  # P(L0) – initial mastery
Self.learn = 0.3  # P(T) – learning rate
Self.slip = 0.1   # P(S) – slip probability
Self.guess = 0.25 # P(G) – guess probability

Def update_mastery(self, concept_id, is_correct, time_since_last):
# Apply knowledge decay
Decay_factor = math.exp(-0.01 * time_since_last)
Current_mastery = self.get_mastery(concept_id) * decay_factor

# Update based on response
If is_correct:
Posterior = self.bayes_update_correct(current_mastery)
Else:
Posterior = self.bayes_update_incorrect(current_mastery)

Self.store_mastery(concept_id, posterior)
Return posterior

# 3. Database Schema
CREATE TABLE student_knowledge (
Student_id UUID REFERENCES users(id),
Concept_id UUID REFERENCES concepts(id),
Mastery_probability DECIMAL(5,4),
Confidence_interval DECIMAL(5,4),
Last_updated TIMESTAMP,
Observations_count INTEGER,
PRIMARY KEY (student_id, concept_id)
);

# 4. API Endpoint
@router.post(“/api/v1/adaptive/update-knowledge”)
Async def update_student_knowledge(
Student_id: UUID,
Concept_id: UUID,
Response: StudentResponse
):
Tracer = BayesianKnowledgeTracer()
New_mastery = tracer.update_mastery(
Concept_id,
Response.is_correct,
Response.time_since_last
)
Return {“mastery”: new_mastery, “next_lesson”: tracer.recommend()}
```
