## Atomic Prompt Standard — Mandatory Compliance
Every prompt in this Worksheet satisfies all five Atomic conditions:
- Single Operation — performs exactly ONE logical action
- Verifiable Input — defines explicit input parameters (type, format, constraints)
- Verifiable Output — defines explicit expected output (type, format, success criteria)
- Zero Compound Logic — no AND ALSO clauses; no multi-step instructions in one prompt
- Dependency Declared — lists any prerequisite prompts that must execute first
